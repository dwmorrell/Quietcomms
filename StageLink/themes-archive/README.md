# themes-archive

Theme design records that aren't compiled into `theme.h` / flashed to a
board: replaced palettes, and the preview mockups + generator scripts used
to get a theme approved before it went into `THEMES[]`.

- `cyberpunk-legacy.h` — the original generic "Cyberpunk" entry, replaced
  2026-07-15 by "Cyberpunk 2077" in `theme.h`.
- `cyberpunk-2077-preview.png` / `cyberpunk-2077-preview-gen.ps1` — the
  mockup approved before building the Cyberpunk 2077 theme (palette +
  single-button carousel home screen), and the script that renders it.
